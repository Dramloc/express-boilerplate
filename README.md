# express-boilerplate

## Usage

Start local development server (NODE_ENV !== production) or production server (NODE_ENV === production)

```shell
yarn start
```

Serve local development server

```shell
yarn dev
```

Launch production server

```shell
yarn serve
```

Run tests

```shell
yarn test
```

Lint sources

```shell
yarn lint
```
